◆ getAIStrategyMove()

String getAIStrategyMove ( String currentPlayerMove)

Generates a move based on the AI strategy using the last player move.

Parameters
currentPlayerMoveThe current move of the player.
Returns
The AI's strategic response to the player's move.

If it's the first round, the AI chooses a random move. For subsequent rounds, it chooses a move that beats the player's last move.

◆ loop()

void loop ( )

Main loop to listen for game mode and moves from the client and respond with the game result.

This loop reads the game mode and the moves from the client, plays the appropriate game mode, and sends the result back to the client. The client is expected to send the mode followed by the player moves.
